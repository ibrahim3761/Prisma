# Prisma

A minimal TypeScript starter project for learning [Prisma ORM](https://www.prisma.io/) with PostgreSQL. It sets up a simple blog-style schema (`User` and `Post` models) and demonstrates basic create and query operations through Prisma Client.

This repo doubles as my personal Prisma practice space.

## Tech Stack

- **TypeScript**
- **Prisma ORM 7** (`prisma`, `@prisma/client`)
- **PostgreSQL** via the `@prisma/adapter-pg` driver adapter and `pg`
- **tsx** for running TypeScript directly
- **dotenv** for environment variable management

## Data Model

```prisma
model User {
  id    Int     @id @default(autoincrement())
  email String  @unique
  name  String?
  posts Post[]
}

model Post {
  id        Int     @id @default(autoincrement())
  title     String
  content   String?
  published Boolean @default(false)
  author    User    @relation(fields: [authorId], references: [id])
  authorId  Int
}
```

A `User` can have many `Post`s, and each `Post` belongs to one `User`.

## Getting Started

### Prerequisites

- Node.js
- A PostgreSQL database (you can spin up a free hosted instance with `npx create-db`, as suggested in the schema file)

### Setup

1. Clone the repository and install dependencies:
   ```bash
   git clone https://github.com/ibrahim3761/Prisma.git
   cd Prisma
   npm install
   ```

2. Create a `.env` file in the project root with your database connection string:
   ```env
   DATABASE_URL="postgresql://user:password@host:port/database"
   ```

3. Run the migrations to create the `User` and `Post` tables:
   ```bash
   npx prisma migrate dev
   ```

4. Generate the Prisma Client (if not already generated by the migration step):
   ```bash
   npx prisma generate
   ```

### Running the Example Script

`src/script.ts` creates a new post and then fetches all users along with their posts:

```bash
npx tsx src/script.ts
```

## Notes

- The Prisma Client is generated into `generated/prisma` (configured in `schema.prisma`) rather than the default `node_modules/.prisma` location.
- The `@prisma/adapter-pg` driver adapter is used to connect Prisma Client to PostgreSQL through the `pg` driver.
- `src/server.ts` is currently a placeholder for future server-side code.
